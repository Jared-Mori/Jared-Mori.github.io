Alien Escape Game

Description: This project is the final project from the class Intro to Game Design. It utilizes the Unity engine, and is a game centered around escaping increasingly larger mazes, while fighting off alien attackers.

Technologies Used: The project utilizes the Unity engine. Within Unity, it utilizes an external NavMesh package to improve enemy AI. It also utilizes C# in scripts.

Challenges: The two hardest challenges faced in this project were enemy AI and map generation. Enemy AI utilizes an external package to allow for 2D rendering of Unity's NavMesh. Map generation utilizes a backtracking algorithm to generate a complete maze, which is randomized every level up.

Outcome: The final result is an unpolised game meant so showcase skills in Unity and C#. The game utilizes a number of interesting and complex algorithms to achieve its effects, and demonstrates the ability to utilize much of Unity's toolkit.
